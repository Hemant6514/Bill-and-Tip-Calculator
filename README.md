# 💸 Tip Calculator App

A modern and responsive Tip Calculator built using HTML, CSS, and JavaScript.  
This project helps users quickly calculate the total bill, tip amount, and split the bill per person with a clean glassmorphism UI and smooth interactions.

---

## 🚀 Features

- Calculate total bill with tip
- Split bill between multiple people
- Preset tip percentage buttons
- Custom tip percentage input
- Interactive active button states
- Smooth hover animations
- Glassmorphism UI design
- Fully responsive mobile-first layout
- Modern gradients and neon shadows
- Dynamic greeting message after calculation

---

- Glassmorphism Design
